# Finance Factors Dashboard - Next.js Application

> **📖 For complete documentation, see the [main README](../README.md)**

This directory contains the Next.js 15 dashboard application with interactive Chart.js visualizations and live API integration. The project features **automatic deployment** to Vercel via GitHub Actions.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Open http://localhost:3000
```
